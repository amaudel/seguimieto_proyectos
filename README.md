# Seguimiento de Proyectos - Cooperativa

Este proyecto es una plataforma web para el seguimiento, control y gestión de proyectos dentro de la cooperativa. Permite a los administradores y miembros monitorear el avance, presupuestos, hitos e impacto de cada iniciativa en tiempo real.

## 🚀 Características Principales

*   **Gestión de Proyectos**: Creación, edición y visualización del ciclo de vida de los proyectos de la cooperativa.
*   **Control Financiero**: Monitoreo de presupuestos asignados, ejecutados y desviaciones.
*   **Hitos y Tareas**: Planificación y seguimiento del progreso de actividades clave.
*   **Roles y Permisos**: Accesos diferenciados para administradores, líderes de proyecto y visualizadores.
*   **Reportes y Dashboards**: Indicadores clave de rendimiento (KPIs) y gráficos visuales sobre el estado de la cartera.

## 🛠️ Stack Tecnológico Propuesto

*   **Frontend**: HTML5, Vanilla CSS (diseño premium, responsivo y dinámico) y JavaScript ES6+.
*   **Base de Datos**: PostgreSQL / SQLite (según el entorno).
*   **Documentación**: Markdown con diagramas Mermaid integrados.
